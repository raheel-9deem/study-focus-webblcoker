**Solutions**:
1. Make sure you selected the correct folder

### Can't See Icon in Toolbar

**Symptoms**: Extension loaded but no icon visible

**Solutions**:
1. Click the puzzle icon (ğŸ§©) in toolbar
2. Find "Study Focus â€“ Distraction Blocker"
3. Click the pin icon next to it
4. Brain icon now appears in toolbar

---

### Websites Not Blocking

**Symptoms**: Can still access blocked sites during focus

**Solutions**:
1. Check status is "Focused" (green)
2. Refresh the page after starting focus
3. Verify website is in blocked list
4. Check website format (just domain, no http://)

---

### Timer Not Counting Down

**Symptoms**: Timer stuck at initial time

**Solutions**:
1. Close and reopen popup
2. Try stopping and restarting
3. Reload extension in `chrome://extensions/`
4. Check Chrome console for errors

---

### No Notification

**Symptoms**: Timer ends but no alert

**Solutions**:
1. Check Chrome notification permissions
2. System Settings â†’ Notifications
3. Ensure Chrome notifications enabled
4. Try test notification in Chrome
