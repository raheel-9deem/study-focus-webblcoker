**Why**: Save your blocked websites list and timer state
**Privacy**: All data stays on your device, never sent anywhere

### ‚úì Alarms
**Why**: Keep timer running even when popup is closed
**Privacy**: Just sets local timers, no data collection

### ‚úì Notifications
**Why**: Alert you when focus session completes
**Privacy**: Local notifications only, no external services

### ‚úì Tabs
**Why**: Communicate with web pages to block sites
**Privacy**: Only checks URLs, doesn't read page content

### ‚úì All URLs (host permissions)
**Why**: Check if current website should be blocked
**Privacy**: Only compares domain names, doesn't access page data

## üêõ Troubleshooting

### Extension Won't Load

**Problem**: Error message when loading
**Solution**:
1. Make sure you selected the correct folder
2. Check all files are present
3. Verify manifest.json has no typos

### Icon Not Showing

**Problem**: Can't see extension in toolbar
**Solution**:
1. Click puzzle icon (üß©)
2. Find "Study Focus"
3. Click pin icon to pin it

### Websites Not Blocking

**Problem**: Can still access blocked sites during focus
**Solution**:
1. Check that status shows "Focused" (green)
2. Try refreshing the page
3. Make sure site is in blocked list

### Timer Not Working

**Problem**: Timer doesn't count down
**Solution**:
1. Close and reopen popup
2. Make sure Chrome has notification permissions
3. Try restarting Chrome

### Notification Not Showing

**Problem**: No alert when timer finishes
**Solution**:
1. Check Chrome notification settings
2. System Settings ‚Üí Notifications
3. Ensure Chrome can send notifications
